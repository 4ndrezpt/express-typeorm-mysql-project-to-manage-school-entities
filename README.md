## API made with ExpressTs, TypeORM and MySql.

In this API I have implemented a backend to manage information between and API entrypoints and MySql database where we can find a project making relations between students, courses, students_x_course,  and teachers using TypeORM.

### Technologies used in this project

1. Node.
2. Express.
3. Typescript.
4. TypeORM.
5. MySQL

###Other dependencies 
* Morgan.
* Cors.
* dotenv.
* Mysql2.
